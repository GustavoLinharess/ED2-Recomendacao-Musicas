# 🎵 Sistema de Recomendação de Músicas

Projeto desenvolvido para a disciplina de **Estruturas de Dados II** — Universidade de Brasília (UnB).

O sistema utiliza um **grafo bipartido** para conectar usuários e músicas, com o objetivo de recomendar faixas com base nas avaliações dos usuários.

> 🚧 Projeto em desenvolvimento

---

## 🚀 Como Executar

**Pré-requisito:** Python 3.10+

```bash
git clone https://github.com/GustavoLinharess/ED2-Recomendacao-Musicas.git
cd ED2-Recomendacao-Musicas
python main.py
```
